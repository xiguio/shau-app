# 鲨鱼CMS客户端

基于dcloud uniapp开发的一款现代内容管理平台，可同时编译至各大平台小程序、iOS和安卓APP。

### 服务端

需结合服务端使用，服务端请移步 [shau-server](https://github.com/xiguio/shau-server)

### 功能模块
- [x] 内容列表
- [x] 内容详情
- [x] 微信登录
- [x] 每日签到
- [x] 用户等级
- [ ] 邀请好友
- [ ] 帮助中心
...更多功能策划中
